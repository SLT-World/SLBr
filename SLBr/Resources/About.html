SLBr

The browser that prioritizes a faster web
Browse the web with a fast, lightweight web browser.

Based on CefSharp

CefSharp is an embedded Chromium browser for .NET apps such as WPF and Winforms. It is a lightweight .NET wrapper for the Chromium Embedded Framework (CEF).
CefSharp follows modern web standards, and supports HTML5, JavaScript, CSS3, HTML5 audio/video elements as well as WebGL. CefSharp has basically everything a modern browser would need.

SLBr's most notable features

Feature Description
Tab Unloading Inactive/unused but initialized browser processes are automatically unloaded.
Switchable render modes Switch to the most suitable render mode for improved rendering.
Built-in Tracker Blocker & Ad Blocker Blocks intrusive ads and trackers.
Parallel downloading Downloads multiple chunks at the same time to accelerate the downloading of large files.
Google Weblight Servers Load optimized webpages through Google Weblight servers.
Multi modes Private mode (Incognito), Developer mode (Experimental features), Chromium mode (Default Chromium), Internet Explorer mode (Internet Explorer).
World Wide Web alternatives IPFS, Gemini & Gopher view.

System Requirements

  • Microsoft Visual C++ 2019 Runtime
  • .NET 6.0
  • Segoe MDL2 Assets
  • Windows 7 and above

Keyboard Shortcuts

  • F5 (Refresh)
  • Ctrl+S (Screenshot)
  • F12 (Developer Tools)

Limitations

  • Only open source media codecs are supported. Proprietary codecs such as H.264 and AAV are not supported due to licensing requirements
  • Only on Desktop, Laptop & Tablet (Desktop mode)
  • Windows OS only
  • Limited extension APIs
  • No proxy support
