What's New in SLBr 2022.3.20.0

Every update of SLBr delivers many new features.

Note from the developer: Conducted a lot of test runs this time and hopefully much of what's needed to be fixed are fixed.

UI Changes

  • A status bar
  • A finished downloading prompt
  • Renamed the context menu button "Close menu" to "Cancel" and moved it to the end of the menu
  • Open URL in new tab context menu
  • Opens About page on first try
  • Fixed various errors that caused crashes on first try
  • Rearranged Tools Menu

Bug fixes

  • SLBr will render properly on Intel i5 processors, SLBr takes a giant leap forward (For developers who are wondering how, here's the line of code that solved it: "RenderOptions.ProcessRenderMode = RenderMode.SoftwareOnly;")
  • Fixed prompt not closing problem (Binding problem)

Code Changes

  • "javascript:" scheme added
  • More details added to the About Page
  • Moved and shortened code
